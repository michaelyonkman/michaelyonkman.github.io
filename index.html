My Story

I started my career working in the luxury retail industry at Barneys New York. I really enjoyed my work and my colleagues and built many strong client relationships. I was able to use my visual sense and design eye to select products for the store and for my clients. As the industry started to shift toward online commerce and away from brick and mortar, I started to feel like I needed a change of pace and a new challenge. I've always had a love of technology and remembered how much I enjoyed styling my MySpace page back when that was still a thing. I started doing some self study and discovered I had a passion for coding. Being able to manipulate the visual element was really exciting to me. I get the same rush from solving a difficult coding challenge that I used to get from having a really great sales day. When Barneys closed I decided to take the leap and joined Fullstack Academy's immersive software engineering program. I'm looking to join a team that communicates and maintains a positive and respectful attitude even when they face challenges and fosters learning and continued growth.

Projects

Ahoy!

A mobile scavenger hunt application built with React Native and Expo. The app employs photo recognition from Clarifai, cloud image storage from Cloudinairy, geolocation data storage on a PostgreSQL database using PostGIS, and push notifications to make in-game friends.

FullTrack

A single-page web application built with React to display live transit data from Chicago’s CTA API. The app utilizes web sockets and Streamdata.io to access real time CTA transit data and a map component using Leaflet.js and React-Leaflet to display train and bus locations.

  • Github

Douglas Fir Apothecary

An e-commerce web application built with React featuring a RESTful backend using Express and Sequelize, a product catalog, persistent user sessions, and an integrated checkout process with Stripe for payment processing.

  • Github

Recipe Roost

A single-page web application to host all of your favorite recipes in one place. Built with React and React Bootstrap for a responsive experience on both desktop and mobile platforms. The app features a relational database utilizing PostgreSQL for searching and sorting results based on recipe name, category, and ingredients.
